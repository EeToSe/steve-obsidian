## 1. Mechanical Unlocking

流程逻辑说明：

1. **地面阶段 (1-4)**：先松动脊柱，再激活核心与臀部。

2. **弓步阶段 (5)**：从地面过渡到站立的连接动作。

3. **站立阶段 (6)**：最后进行动态摆腿，模拟深蹲/硬拉的实际发力模式。

| **Exercise (Dosage)**                           | **Target & Details**                                                                  |
| ----------------------------------------------- | ------------------------------------------------------------------------------------- |
| **1. Cat-Cow**<br><br>(10-15 reps)              | **脊柱波浪：温和唤醒脊柱，建立呼吸节奏。**<br><br>吸气时胸椎伸展（塌腰），呼气时骨盆后倾（拱背）。                               |
| **2. Sphinx Pose**<br><br>(8-10 reps, hold 3s)  | **胸椎伸展：纠正圆肩，针对性激活上背部，避免腰椎代偿。**<br><br>前臂撑地，小臂用力向后“拉”地面，将胸骨向前上方顶出。保持下巴微收，感受上背部的伸展。     |
| **3. Open Books**<br><br>(8-10 reps / side)     | **胸椎旋转：解锁胸椎灵活性，优化推举和深蹲时的背部体态。**<br><br>侧卧，膝盖压住地面（或泡沫轴）以锁定骨盆，像翻书一样打开手臂，视线跟随手掌移动。       |
| **4. Glute Bridge**<br><br>(15 reps, 1s at top) | **臀部激活：唤醒臀大肌，稳定骨盆，主动放松髋前侧紧绷。**<br><br>脚后跟发力蹬地，在顶端用力夹紧臀部停留1秒。注意不要靠“拱腰”把身体顶起来，核心要收紧。    |
| **5. Greatest Stretch**<br><br>(5 reps / side)  | **一站式解药：整合全身关节，大幅度拉伸髋部与腿后侧。**<br><br>此时身体已热，可尝试更大活动度。若后腿髋前侧依然紧绷，可做“后脚抬高”变式。           |

## 2. The Flush 

**主选方案: 跑步机爬坡 Incline Walking**

- 时长: 20–25 分钟
    
- 坡度: 8–12 %
    
- 速度: 3.5–4.5 km/h
    
- 心率目标区: 约 120–130 次/分钟
    

**替代选项 (高疲劳日优先考虑)**

- 固定自行车低阻力骑行
    
- 原地踏步配合有节奏摆臂
    

**统一原则**

- 不扶扶手, 依靠核心和臀部维持身体稳定
    
- 手掌朝前、拇指朝外摆臂, 强迫肩关节外旋, 打开胸廓
## 3. Restoration

### 3.1 软组织质量管理 (泡沫轴 / 筋膜球)

建议总时间约 5 分钟, 针对当天“最紧”的区域处理。

1. **胸椎滚压 (Thoracic Extension)**
    
    - 泡沫轴横放在上背部, 双手抱头或抱胸
        
    - 只在肩胛骨区域上下滚动, 不要压到腰椎
        
    - 出现轻微“咔咔”声属于关节调整, 一般属正常现象
        
2. **臀部 / 梨状肌 (Glutes / Piriformis)**
    
    - 坐在泡沫轴上, 一腿搭在对侧膝上做“4 字型”
        
    - 身体略向一侧倾斜, 慢慢寻找酸痛触发点
        
    - 对缓解下背部紧张和坐姿导致的臀部僵硬很有帮助
      ![[臀部-泡沫轴.png]]
1. **阔筋膜张肌 / 髂胫束外侧 (TFL / IT Band 区域)**
    
    - 侧卧在泡沫轴上, 从髋外侧到膝外侧缓慢滚动
        
    - 过程会比较酸痛, 但对减轻膝外侧疼痛、优化深蹲力线有重要价值
	    ![[阔筋膜-泡沫轴.png]]

### 3.2 静态拉伸: 组织延长

每个动作保持 60–90 秒以上, 才有机会触发 GTO 抑制, 真正“放松而不是拉紧”。

1. **Couch Stretch + Dead Hang**

	沙发拉伸步骤（90秒->2m）:
	
	- 选择墙或沙发边
	    
	- 后侧膝盖跪在软垫上, 尽量靠近墙根
	    
	- 小腿贴墙, 脚背抵住墙面
	    
	- 上身逐渐竖直, 同侧臀部主动收紧
	    
	- 轻微收下腹, 避免腰椎过度前凸
    
	Dead Hang 简述:
	
	- 双手正握单杠, 放松身体自然下垂
	    
	- 保持肩胛稳定不过度耸肩
	    
	- 可分 2–3 组, 每组 20–40 秒, 视握力与肩关节舒适度调整
    

2. **门框胸肌拉伸 (Doorway Pec Stretch)**
	- 一侧前臂或上臂放在门框上
	    
	- 身体微微向前和向外旋转, 感受胸大肌和胸小肌被拉长
	    
	- 目标是改善圆肩、打开前侧胸廓, 减少第二天推举时肩峰撞击风险

3. **鸽子式 (Pigeon Pose)**
	- 针对髋外旋肌群和臀部深层肌肉
	    
	- 前腿屈髋外旋放在身前, 后腿伸直放在身后
	    
	- 身体向前缓慢俯身, 找到可承受的牵拉感
	    
	- 对减轻下背部由髋僵硬带来的代偿压力很有帮助

## 4. The Fork

训练恢复后, 根据接下来的安排选择 A 或 B。

### 4.1 选项 A: 无午睡, 直接回实验室工作

**A1 爆发式唤醒 (以上肢或全身轻负荷为主)**

- 快速俯卧撑: 1 组 8–10 次, 保持动作干净, 不必力竭
    
- 俯身弹力带划船: 1 组 15 次, 重在“快而有控制”, 感受上背发力
    

**A2 呼吸 + 冷刺激组合**

- 冷水洗脸约 10–20 秒
    
- 呼吸模式: 吸气约 4 秒,屏息 1–2 秒,用力呼气 4–6 秒
        
- 重复 3 轮, 用来提神、清理残余的“困倦感”

### 选项 B 若准备午睡 (Nap)

采用 Box Breathing 箱式呼吸 2 分钟: 吸气4秒 -> 屏息4秒 -> 呼气4秒 -> 屏息4秒，连续 8–10 个循环。
如有午睡后难以醒来的情况：在躺下前喝半杯咖啡（约 80–120 mg 咖啡因 ）
