> **核心哲学：** 顺应大脑能量曲线，而非对抗它。管理能量，而非仅仅管理时间。

## 🌅 The Wind-Up (06:30 - 07:30)

**—— Biological Priming**

- **💧 Hydrate:** 起床即喝一杯水（激活大脑）。
    
- **🧘 Mindful Moment:** 5-10 分钟冥想或深呼吸（清除杂念，提升前额叶控制力）。
    
- **🤸 Light Movement:** 轻度拉伸、瑜伽或散步（促进血液流向大脑）。
    
- **🚫 No-Wifi Zone:** **严禁**查看邮件/社交媒体。此时大脑是被动反应模式，一旦接收外界信息，Deep Work 的“主动进攻”状态即刻瓦解。
    

## 🚀 Morning Deep Work I (07:30 - 10:30)

**—— The "Eat the Frog" Session**

### 1. Core Principles (Hunter Mode)

- **生理状态:** 前额叶皮层活跃度巅峰，逻辑与决策能力满格。
    
- **目标:** 攻克科研中最难的骨头（推导公式、撰写 Methodology、分析核心数据）。
    

### 2. How to Cheat Your Brain

- **Zero Friction:** 昨晚已预下载好论文、预配置好代码环境。
    
- **Warm-Up (5-10min):** 如果主任务太难，不要硬啃。
    
    - _动作:_ 重读昨天的笔记、读一段相关的 Paper、整理一下变量名。
        
    - _目的:_ 骗过杏仁核的恐惧反应，平滑进入心流。
        
- **Process over Outcome:** “现在的任务不是‘解决这个问题’，而是‘专注执行这 50 分钟’。”
    

### ✅ The High-Focus Checklist (Start Here)

_在开始 Deep Work 前，快速过一遍：_

- [ ] **干扰屏蔽:** 手机静音/移出房间？不相关的网页/Tab 关闭了？
    
- [ ] **单核模式:** 这里的 3 小时只做这一件事 (Single-tasking)？
    
- [ ] **任务拆解:** 我是否清楚今早具体要解决哪个小问题？（例如：不是“写论文”，而是“写完 Method 部分的第二段”）。
    

### 3. Action Plan: 50/10 Blocks

- **07:30 - 07:35 [Setup]:** 📝 在便签写下本时段只解决 **具体问题X**。
    
- **07:35 - 08:25 [Block 1 - Heavy Lift]:** **深度任务:** 涉及高认知负荷的工作（写作、编码、推导）。
    
- **08:25 - 08:35 [Break]:** 10分钟完全离屏幕，喝水、远眺。
    
- **08:35 - 09:25 [Block 2 - Flow]:** **延续任务:** 保持惯性。
    
- **09:25 - 09:35 [Break]:** 动态拉伸。
    
- **09:35 - 10:25 [Block 3 - Sprint]:** **收尾冲刺:** 完成这一阶段的里程碑。
    

### 📈 Morning Momentum Log

- **胜利时刻:** 攻克了哪个具体的理论卡点？ __________
    
- _Check:_ 方向对吗？是否需要根据新发现调整后续计划？
    

## ⚔️ Afternoon Deep Work II (14:00 - 16:30)

**—— The "Output Flow" Session**

### 1. Core Principles (Builder Mode)

- **生理状态:** 适合“机械性输出”或“实验执行”。
    
- **目标:** 将想法转化为可视化结果（代码实现、图表绘制、初稿堆砌）。
    

### 2. How to Cheat Your Brain

- **Shitty First Draft:** 允许糟糕的初稿。
    
- **Batching:** 批量处理低能耗任务（如回邮件、填表）。
    

### 3. Action Plan: 30/5 Speed Runs

- **13:30 - 13:35 [Setup]:** 规划 Sprint 目标（e.g., "跑通测试用例 A/B/C"）。
    
- **13:35 - 14:05 [Cycle 1]:** 30分钟狂写代码/输出。
    
- **14:05 - 14:10 [Pause]:** 5分钟深呼吸。
    
- **... (重复循环)**
    
- **Long Break:** 完成 4 个循环后 (~2h)，进行长休息。
    

## 🌙 Phase 5: Evening Deep Work III (18:45 - 21:00)

**—— The "Synthesize & Reset" Session**

### 1. Core Principles

- **生理状态:** 意志力耗尽，发散思维活跃。
    
- **目标:** Cleanup, Connect, Close - Pave the way for tomorrow.
    
- **禁忌:** 🚫 禁止推导复杂公式、Debug 极难报错。
    

### 2. Modular Execution

_将数字资产归档与生活琐事处理相结合。_

#### 🔹 模块 A: Soft Input - [18:45 - 19:45]

1. **Tab Bankruptcy (网页标签清理):**
    
    - 有用且需深读 → Obsidian clippings； reference → Zotero.
        
2. **LLM Chat Review (AI 对话挖掘):**
    
    - 提取有价值的 Prompt 或 Snippets 归档至 Obsidian。
        
3. **Scanning (泛读):**
    
    - 快速浏览 3-5 篇论文摘要，筛选明天需精读的。
        
4. **Big Picture:**
    
    - 拿出一张白纸，随意画一下目前的科研架构图，利用晚上的发散思维寻找遗漏的逻辑。
        

#### 🔹 模块 B: Low-Battery Execution - [19:45 - 20:30]

_适合：格式调整、数据清洗、并不费脑的重复劳动_

1. **Beautification (美化):** 调整 LaTeX 格式、画 PPT 图表。这种工作有明确的视觉反馈，能带来多巴胺。
    
2. **Data Prep (预备):** 清洗数据，把明早要跑的代码环境配好。
    
3. **Admin Zero:** 集中处理微信/邮件/购物快递/notifications/AI news/X bookmarks，清空大脑RAM。
    

#### 🔹 模块 C: Shutdown Ritual - [20:30 - 21:00]

1. **Define the Frog:** 明确明早的**唯一**核心任务。
    
2. **Environment Reset:** 清理桌面，打开明日文档和代码。
    
3. **Digital Sunset:** 合上电脑，宣布今日科研结束。

	- **明天的 Frog 是:** ______________________
    

**Supervisor's Note:** 这套系统的关键在于**节奏**。早晨像猎人（聚焦），下午像工匠（输出），晚上像图书管理员（整理）。保持节奏，不要试图全天冲刺。
